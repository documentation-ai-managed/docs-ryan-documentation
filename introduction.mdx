## Overview

Ryan Documentation helps you create, organize, and maintain comprehensive documentation for your projects. You build custom documentation spaces with powerful MDX support, enabling rich, interactive content like code examples, diagrams, and embedded components. Whether you manage a single repository or multiple teams, Ryan streamlines your workflow with intuitive tools for collaboration and publishing.

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://ryan-docs.com` with your email or GitHub account.
  </Step>
  <Step title="New Space" icon="plus">
    Click **New Space** and enter a name like `MyProjectDocs`.
  </Step>
  <Step title="Add Page" icon="file-plus">
    Create `introduction.mdx` and start writing with MDX syntax.
  </Step>
  <Step title="Publish" icon="globe">
    Preview and publish your space to share a live URL.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example: Fetch your space data
  const response = await fetch('https://api.ryan-docs.com/spaces/YOUR_SPACE_ID');
  const space = await response.json();
  console.log(space.pages);
  ```
  ```python
  # Example: List pages in a space
  import requests
  response = requests.get('https://api.ryan-docs.com/spaces/YOUR_SPACE_ID')
  space = response.json()
  print(space['pages'])
  ```
</CodeGroup>
